# POS-Form
Formulir untuk pendaftaran calon staff baru

## Instalasi
**1. edit file includes/config.php**
 ```php
 'webhookURL' => "", //webhookURL
 'username' => "", //username
 'avatarWebhook' => "" //avatar
 ```
**2. Selesai, sisanya tinggal kalian edit buat formnya**
